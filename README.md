# Portfolio

A personal portfolio website showcasing my work and projects.

## Features

- Responsive design
- Modern UI/UX
- Easy to customize

## Files

- `index.html` - Main portfolio page
- `style.css` - Styling
- `script.js` - Interactive features
- `profile.jpg` - Profile image

## Deployment

This portfolio is ready to be deployed on GitHub Pages:

1. Push this repository to GitHub
2. Go to repository Settings → Pages
3. Select "Deploy from a branch"
4. Choose `main` branch and click Save
5. Your portfolio will be live at `https://yourusername.github.io/repository-name`
